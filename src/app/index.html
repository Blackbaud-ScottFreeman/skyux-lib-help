Using the bbHelpOpenOnClick directive

  • Grab By Keyboard
  • anchor link: dc-make-grant-rec.html
  • button:

Setup:

  • Has a bb-help-key through local demo component toggle-key switching between two pageDefaultKey's: dc-explore-grant-opportunities.html and dc-admin-menu.html.

  • Has a bb-help-button with the helpKey dc-make-grant-rec.html.

  • Has a Sky Modal with a bb-help-key with the helpKey dc-fundinfo.html.

Expected Behavior:

  • Invoker/Toggle Widget: Should load the current toggled pageDefaultKey, either dc-explore-grant-opportunities.html or dc-admin-menu.html.

  • Help Button: Should open the dc-make-grant-rec.html help page from bb-help.

  • Invoker/Toggle Widget: (with modal open) Should open the dc-fundinfo.html help page from bb-help.
